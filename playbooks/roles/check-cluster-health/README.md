check-cluster-health
=========

This role can be used to check if the cluster operators and nodes are healthy.

Requirements
------------

- OCP 4.x cluster

Role Variables
--------------

- None

Dependencies
------------

- None

Example Playbook
----------------

```
- name: Include check-cluster-health role
  include_role:
    name: check-cluster-health
```
